## How to use a skill

1. Before building in an area, open its skill file.
2. Follow the **checklist**; avoid the **anti-patterns**; mirror the **examples**.
3. If you discover a better practice, update the skill (Documentation-First).

Each skill file contains: **Best Practices · Checklist · Anti-Patterns · Examples**.
